# Simcard Microservices Architecture

This repository contains the microservices architecture for the Simcard application using Docker.

## Services:
- User Service
- Product Service
- Order Service
- Payment Service

### How to Run
1. Build the services using Docker Compose.
2. Run `docker-compose up` to start all services.

## Folder Structure:
- `user-service/` : Contains the User Service.
- `product-service/` : Contains the Product Service.
- `order-service/` : Contains the Order Service.
- `payment-service/` : Contains the Payment Service.
- `docker-compose.yml` : The Docker Compose file to orchestrate the services.
